# Snake game

Easy snake game as part of a python programming course at the university of Lausanne.

## Installation for MAC users

**1. Install [Python 3.7](https://www.python.org/downloads/release/python-370/) (or a newest version)<br/>**
Go to the official python website, download and install `pyton 3.7`.<br/>

**2. Download and Install [pygame](https://www.pygame.org/wiki/GettingStarted) using the terminal<br/>**
Open a terminal window.<br/>
Copy and paste the follwing line `python3.7 -m pip install Pygame==2.0.1` and hit ENTER.<br/>
This should install `pygame 2.0.1` into the python 3.7 folder you've previously installed, if you've downloaded a newer version of python don't forget to adapt the command line according to it.<br/>
(Python 3.7 should come with the `pip` command, if it doesn't work you'll find more information [here](https://pip.pypa.io/en/stable/installing/))<br/>

**3. Download the code from [github](https://github.com/aurelienHamouti/snake_game)<br/>**
Click on the green button and export the `ZIP-file` containing all the code to your computer.
  
**4. Execute the code<br/>**
Unzip the downloaded file.<br/>
Open the `snake_game` folder and right-click on the `snake_game.py` file.<br/>
Select `open with >` **IDLE 3.7**.<br/>
Once the IDLE window is open, go to the `RUN menu` and select **Run module** or press **F5**.<br/>
The game should start.<br/>

ENJOY!
