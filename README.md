# Snake game

Easy snake game as part of a python programming course at the university of Lausanne.

## Installation

1. Install [Python 3.7](https://www.python.org/downloads/release/python-370/) (or a newest version)

2. Download and Install [pygame](https://www.pygame.org/wiki/GettingStarted)

3. Download the code 
  Download the code to your computer.
  
4. Execute the code
Unzip the downloaded file.
  For windows users : open the snake_game folder and execute the snake_game.py file by double clicking on it to start the game,
  
  For mac users : open a terminal window and change the directory to the snake_game folder
    An easy way to do this is to right-click on the file, while in the rignt-click menu, hold the `option` key and select copy "snake_game" as Pathname
    Type cd /Pathname in the terminal window and hit Enter
    Type python ./snake_game.py
